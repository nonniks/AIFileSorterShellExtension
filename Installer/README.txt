============================================================
             AI File Sorter - Installation Guide
============================================================

This package contains the AI File Sorter shell extension that
helps you automatically organize files in folders using AI.

INSTALLATION INSTRUCTIONS:
--------------------------
1. Right-click on Install.ps1 and select "Run with PowerShell"
2. If prompted about execution policy, choose "Yes" to proceed
3. The installer requires administrator privileges
4. Follow the on-screen instructions

AFTER INSTALLATION:
------------------
1. Right-click on any folder or in the background of a folder
2. Select "Sort Files" from the context menu
3. The extension will use AI to organize files into categories

FIRST USE:
---------
1. You'll need to provide an OpenRouter API key
2. Get a free key from: https://openrouter.ai/
3. Enter this key when prompted by the extension

UNINSTALLATION:
--------------
1. Right-click on Uninstall.ps1 and select "Run with PowerShell"
2. The uninstaller will remove all components from your system

TROUBLESHOOTING:
---------------
If the context menu option doesn't appear after installation:
1. Restart your computer
2. Try running the installer again

CONTENTS:
--------
- AIFileSorterShellExtension.dll - Main application
- SharpShell.dll - Required dependency
- Newtonsoft.Json.dll - Required dependency
- Install.ps1 - Installation script
- Uninstall.ps1 - Uninstallation script
- README.txt - This file
